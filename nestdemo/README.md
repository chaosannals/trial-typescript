# nest 示例

相关：
trial: js
exert: vue

## 生成指令

```bash
# 生成模块
nest g mo talk

# 生成网关
nest g ga talk
```

## Running the app

```bash
# 开发
$ npm run start

# 监听模式 watch
$ npm run start:dev

# 发布模式
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
